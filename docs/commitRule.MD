TEMPLATE COMMIT MESSAGE CHUẨN

1. Commit dạng tính năng
   feat: mô tả ngắn gọn tính năng

- Giải thích chi tiết thay đổi
- Lý do / mục đích
- Ảnh hưởng (nếu có)

2. Commit dạng sửa lỗi
   fix: mô tả lỗi đã sửa

- Mô tả nguyên nhân
- Mô tả cách fix
- Các phần bị ảnh hưởng

3. Commit cho database / schema
   db: tạo mới bảng Users và sửa quan hệ giữa Orders - Discounts

- Tạo bảng mới
- Thêm foreign key
- Thay đổi ràng buộc

4. Commit cho UI/UX
   ui: cập nhật giao diện trang product detail

- Refactor layout
- Tăng kích thước font
- Thêm animation

5. Commit cho cấu hình (config)
   chore(config): setup CI/CD pipeline cho repo

- Thêm GitHub Actions
- Setup build & test

6. Commit cho công việc refactor
   refactor: tối ưu lại logic tính tổng giỏ hàng

- Loại bỏ code thừa
- Tăng readability

7. Commit cho tài liệu
   docs: cập nhật README hướng dẫn chạy backend

- Thêm hướng dẫn migrate DB
- Thêm yêu cầu môi trường

8. Commit cho dependency
   chore(deps): update axios lên version 1.6.0

9. Commit data:
   data: mô tả thông tin ngan gon
